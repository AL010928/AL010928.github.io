Data Visualisation

I am learning data visualization. More specifically (since we are all capable of creating charts in Excel), I am learning how to build single secure channels that link a data visualization output right back to the raw data. This involves learning:

  1. The philosophy behind a more modern approach to data visualization (visualization as a form of data, transparency, reproducibility, Tufte, Albers etc)
  2. How to use Vega-Lite to create charts (I might also explore R later)
  3. How to embed them and style them on a website (and therefore learning HTML, CSS, and JavaScript)

All the visualizations I make are open source — the raw data and the JSON code to create the charts are publicly available on my GitHub.

1. GDP against life expectancy

Both data and chart files are from the Economics Observatory. I simply link to it here.

2. UK Output per hour

This line chart shows UK output per hour from 1997 to 2024. This is my first pass at using Vega-Lite to create a JSON chart file. The data used comes from ONS. Both the CSV (data file) and JSON (chart spec) can be found in my GitHub repo.
